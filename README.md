# Aplikasi Perpustakaan

Aplikasi ini merupakan sebuah sistem manajemen perpustakaan yang dikembangkan dengan JavaFX. Aplikasi ini memanfaatkan sejumlah dependensi untuk mendukung fungsionalitasnya.

## Dependensi

### JavaFX Controls, FXML, Web, dan Media
- **GroupId**: `org.openjfx`
- **ArtifactId**: `javafx-controls`, `javafx-fxml`, `javafx-web`
- **Version**: `21`
- **Deskripsi**: Library JavaFX untuk mengelola tampilan dan interaksi pengguna pada aplikasi.

### ControlsFX
- **GroupId**: `org.controlsfx`
- **ArtifactId**: `controlsfx`
- **Version**: `11.1.2`
- **Deskripsi**: Library tambahan yang menyediakan komponen UI yang lebih kompleks dan fungsionalitas tambahan.

### FormsFX Core
- **GroupId**: `com.dlsc.formsfx`
- **ArtifactId**: `formsfx-core`
- **Version**: `11.6.0`
- **Deskripsi**: Library untuk mempermudah pembuatan dan manajemen formulir dalam aplikasi.

### ValidatorFX
- **GroupId**: `net.synedra`
- **ArtifactId**: `validatorfx`
- **Version**: `0.4.0`
- **Deskripsi**: Library untuk melakukan validasi data pada formulir atau input pengguna.

### Ikonli JavaFX
- **GroupId**: `org.kordamp.ikonli`
- **ArtifactId**: `ikonli-javafx`
- **Version**: `12.3.1`
- **Deskripsi**: Library untuk menambahkan ikon ke dalam aplikasi JavaFX.

### BootstrapFX Core
- **GroupId**: `org.kordamp.bootstrapfx`
- **ArtifactId**: `bootstrapfx-core`
- **Version**: `0.4.0`
- **Deskripsi**: Implementasi Bootstrap di JavaFX untuk menghasilkan antarmuka yang menarik.

### FXGL
- **GroupId**: `com.github.almasb`
- **ArtifactId**: `fxgl`
- **Version**: `17.2`
- **Deskripsi**: Framework permainan yang kuat untuk JavaFX.

### JUnit Jupiter API & Engine
- **GroupId**: `org.junit.jupiter`
- **ArtifactId**: `junit-jupiter-api`, `junit-jupiter-engine`
- **Version**: `5.9.2`
- **Deskripsi**: Library untuk pengujian unit dan integrasi dalam pengembangan proyek.

### MySQL Connector Java
- **GroupId**: `mysql`
- **ArtifactId**: `mysql-connector-java`
- **Version**: `8.0.23`
- **Deskripsi**: Driver JDBC untuk menghubungkan aplikasi Java dengan database MySQL.

## Struktur Proyek

- `src/`: Direktori utama kode sumber.
  - `main/`: Kode sumber utama.
    - `java/`: Kode Java aplikasi.
    - `resources/`: Berkas-berkas pendukung seperti file FXML, gambar, dan CSS.
  - `test/`: Unit test dan pengujian aplikasi.
